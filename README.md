# Fritiner — web pública

Landing y demo interactiva de **Fritiner**, la capa operativa de todo lo que pasa
después de la reserva para empresas de free tours y tours a pie.

- `index.html` — landing
- `demo/index.html` — demo interactiva (datos ficticios de «Luna Tours · Roma»)

Ambos son archivos autocontenidos: sin dependencias externas, funcionan sin servidor.

> Este repositorio contiene **solo** lo publicable. El proyecto completo —documentación
> de estrategia, precios internos y análisis— vive en un repositorio privado aparte y se
> genera aquí con `build-site.sh`.
